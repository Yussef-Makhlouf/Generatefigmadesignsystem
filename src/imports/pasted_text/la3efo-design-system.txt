🔄 INTERACTION STATES & MICRO-INTERACTIONS
- Hover: Card lift (`translateY(-2px)` + shadow increase), Button color shift
- Tap/Click: Scale down 0.98, ripple effect on buttons
- Voting: Animated count increment/decrement, color pulse on active state
- Bookmark: Fill animation, confetti micro-burst on first save
- Loading: Skeleton screens matching exact layout, shimmer animation
- Empty: Fade-in illustration + CTA, subtle bounce on primary button
- Error: Shake animation on invalid form fields, inline error message slide-down
- Success: Checkmark animation, toast slide-up from bottom (mobile) / top-right (desktop)

♿ ACCESSIBILITY & RTL SPECIFICS
- All components: `dir="rtl"` enforced, flex-direction reversed where needed
- Focus indicators: `ring-2 ring-indigo-500/50 ring-offset-2` visible on keyboard nav
- Color contrast: All text ≥ 4.5:1 against background (verify with Stark plugin)
- Screen reader labels: `aria-label` on icon-only buttons, descriptive alt text for images
- Touch targets: Minimum 44x44px for all interactive elements
- Motion: Respect `prefers-reduced-motion`, provide non-animated fallbacks
